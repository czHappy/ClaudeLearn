### 第6题（10分）- 逻辑推理

**正确答案：A**

**解析**：

根据条件逐步推理：

**条件汇总**：
1. 二小的是跳远冠军
2. 一小的不是垒球冠军
3. 甲不是跳高冠军
4. 乙既不是二小的，也不是跳高冠军

**推理**：
- 从条件4：乙不是二小 → 甲或丙是二小
- 从条件1和3：二小是跳远，甲不是跳高 → 假设甲是二小，则甲是跳远 ✓
- 跳远被甲占用，乙不是跳高（条件4）→ 乙是垒球
- 剩下：丙是跳高
- 从条件2：一小不是垒球，乙是垒球 → 乙不是一小
- 乙不是二小（条件4）→ 乙是三小
- 丙是一小

**结果**：
- 甲：二小，跳远
- 乙：三小，垒球
- 丙：一小，跳高

验证选项A：乙是三小的学生 ✓，丙是跳高冠军 ✓

#### **(2) 缓存命中率是多少？（6分）**

**正确答案：D**

**访问过程**：
- 访问A[0]：未命中，加载A[0-7]
- 访问A[1-7]：都命中（共7次）
- 访问A[8]：未命中，加载A[8-15]
- 访问A[9-15]：都命中（共7次）
- 访问A[16]：未命中，加载A[16-23]
- 访问A[17-23]：都命中（共7次）
- 访问A[24]：未命中，加载A[24-31]
- 访问A[25-31]：都命中（共7次）

**统计**：
- 总访问：32次
- 未命中：4次
- 命中：28次

**命中率** = 28 ÷ 32 = 87.5%

---

#### **(3) 访问完数组消耗多少时间？（8分）**

**正确答案：D**

**重要**：每次访问都要先查缓存（1纳秒），如果未命中再访问内存（100纳秒）。

**时间计算**：
- 命中时：查缓存1纳秒
- 未命中时：查缓存1纳秒 + 访问内存100纳秒 = 101纳秒

**总时间**：
```
= 命中次数 × 1 + 未命中次数 × 101
= 28 × 1 + 4 × 101
= 28 + 404
= 432纳秒
```

**答案：D. 432**
