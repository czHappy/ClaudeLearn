# 实验1：条件与循环 - 数字游戏大师

## 实验目标
通过实现一个综合性的数字游戏程序，掌握C++中条件语句和循环结构的使用。

## 知识点清单
- [x] if-else 语句与嵌套条件
- [x] switch-case 语句及其应用场景
- [x] for 循环（传统形式、范围for循环）
- [x] while 与 do-while 循环的区别
- [x] break、continue 与循环控制
- [x] 循环嵌套与复杂度分析
- [x] 条件运算符（三元运算符）

## 实验背景
你需要开发一个"数字游戏大师"程序，包含多个有趣的数字游戏和挑战。这个程序将综合运用各种条件判断和循环结构。

## 实验任务

### 任务1：菜单系统（switch-case）
实现一个游戏主菜单，使用 switch-case 语句处理用户的选择：
- 选项1：猜数字游戏
- 选项2：质数判断器
- 选项3：打印图形
- 选项4：数字分析器
- 选项0：退出程序

要求使用 do-while 循环保持菜单显示，直到用户选择退出。

### 任务2：猜数字游戏（while循环 + 嵌套if）
实现一个猜数字游戏：
- 程序随机生成1-100之间的数字
- 用户有最多10次机会猜测
- 每次猜测后给出"太大"、"太小"或"正确"的提示
- 使用 while 循环控制游戏流程
- 使用嵌套 if-else 判断猜测结果
- 猜对后显示用了几次机会（使用三元运算符判断评价等级）

### 任务3：质数判断器（for循环 + break）
判断一个数是否为质数：
- 读取用户输入的正整数
- 使用 for 循环检查是否有因数
- 使用 break 提前结束循环（发现因数时）
- 特殊处理：1不是质数，2是质数

### 任务4：打印图形（嵌套循环）
打印多种数字图形：
- 直角三角形（数字递增）
- 等腰三角形（使用空格控制对齐）
- 金字塔（嵌套循环 + 条件判断）
- 菱形（上下对称的图形）

要求使用嵌套的 for 循环实现。

### 任务5：数字分析器（综合应用）
分析一个整数的各种特性：
- 计算各位数字之和（while循环提取每一位）
- 判断是否为回文数（比较反转后的数字）
- 判断是否为完全平方数（使用条件判断）
- 统计各位数字中偶数和奇数的个数（使用 continue 跳过某些情况）
- 找出最大和最小的数字位

### 任务6：范围for循环应用
给定一个整数数组，使用范围for循环：
- 遍历数组并输出所有元素
- 计算数组元素的总和与平均值
- 找出数组中的最大值和最小值

## 提示
1. 使用 `rand()` 和 `srand(time(0))` 生成随机数
2. 注意循环的边界条件，避免死循环
3. 合理使用 break 和 continue 优化代码
4. 嵌套循环时注意缩进，保持代码可读性
5. 三元运算符适合简单的条件赋值
