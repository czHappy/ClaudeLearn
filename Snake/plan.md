# 贪吃蛇游戏开发计划

## 技术栈选择

### 前端技术
- **HTML5 Canvas**: 用于游戏渲染，性能优秀，适合2D游戏
- **原生 JavaScript (ES6+)**: 无需依赖框架，轻量级，易于理解
- **CSS3**: 实现美观的UI界面和动画效果

### 设计风格
- **现代简约风格**: 使用渐变色、阴影和圆角
- **色彩方案**:
  - 背景: 深色渐变 (#1a1a2e → #16213e)
  - 蛇身: 霓虹绿色渐变 (#00ff88 → #00cc6a)
  - 食物: 霓虹红色/金色 (#ff0066 → #ff6b9d)
  - 网格: 半透明白色线条
- **动画效果**: 平滑的蛇移动、食物闪烁、分数跳动

## 功能特性

### 核心功能
1. 经典贪吃蛇玩法
2. 键盘控制（方向键 + WASD）
3. 计分系统
4. 速度递增机制
5. 碰撞检测（墙壁和自身）

### 增强功能
1. 开始/暂停/重新开始
2. 最高分记录（localStorage）
3. 音效开关（可选）
4. 响应式设计
5. 游戏速度等级显示

## 游戏参数设置

- 网格大小: 20x20 个单元格
- 每个单元格: 25px x 25px
- 初始蛇长: 3 节
- 初始速度: 150ms/帧
- 速度递增: 每吃5个食物加快10ms

## 美观设计重点

1. **渐变背景**: 深色太空主题
2. **霓虹效果**: 蛇身和食物使用发光效果（box-shadow）
3. **平滑动画**: CSS transition 和 Canvas 动画结合
4. **粒子效果**: 吃到食物时的爆炸效果（可选）
5. **字体**: 使用现代无衬线字体（如 'Orbitron' 或 'Press Start 2P'）
